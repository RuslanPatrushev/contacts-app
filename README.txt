contacts-app
Приложение личного кабинета

Запуск приложения:
Откройте свой редактор кода( WebStorm, VS code или т.п).
Запустите терминал в корневой папке приложения. Выполните:
json-server --watch db.json
Запустите второй терминал в корневой папке приложения. Выполните:
npm start
Откройте приложение на locale-host'е (http://localhost:4200/)

Описание приложения:
В приложении две страницы - страница входа (login) и страница со списком контактов (contacts).
Для реализации авторизации используются запросы с моковыми данными.
https://github.com/typicode/json-server
Страница контактов пользователя доступна только после авторизации
На странице контактов есть возможность добавлять,удалять,редактировать контакты, а также наличие функции поиска контакта по имени.
В разработке используется фреймворк Angular.
